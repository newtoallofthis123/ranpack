    • RanPack is a customizable rainmeter skin. It includes over 10 different usefull widgets like a:

  • Battery Monitor
  • Digital Clock
  • Fancy Time and Date Clock
  • Launcher
  • Quick Links
  • Title
  • Icon
  • System Monitor
  • Network Monitor
  • Disk Usage Statistics
  • Recycle Bin Manager

    • Ranpack also comes with a random joke widget which generates random jokes when you run the rainmeter_joke.exe included in the releases page. After running rainmeter_joke.exe, refresh the widget for a new joke.

    RanPack is very customizable and doesn't come with a defalut layout. It is up you. You can take inspiration from the provided examples.

    Installation

    RanPack comes as a normal rmskin and a full zip file with the rmskin and the rainmeter_joke.exe. You can also get the direct skins by just cloning the repository.
